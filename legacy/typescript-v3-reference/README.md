# Legacy TypeScript Reference

This directory is historical reference code from the pre-v5 TypeScript implementation.

The supported Officegen v5 runtime is the Rust native binary built from the repository root `src/` tree and distributed through GitHub Release assets. The code in this directory is not the default CLI runtime, is not exposed through a root `bin` entry, and should not be used as the supported install path.

Keep this directory only for archaeology, parity research, or deliberate migration comparisons. New runtime work should go into the Rust implementation.
