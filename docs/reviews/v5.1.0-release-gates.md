# Officegen-CLI v5.1.0 Release Gates

This checklist is the publish blocker for v5.1.0. Do not create the v5.1.0 tag, upload assets, or announce the release until the required gates pass for the exact release commit and version.

Required release commands:

```bash
npm run version:bump -- 5.1.0
npm run version:check
npm run installer:smoke
cargo fmt --check
cargo test --locked
cargo build --release --locked
npm test
npm run build
npm run typecheck
npm run v5:acceptance -- --bin target/release/officegen --expected-version 5.1.0
npm run v5:acceptance -- --bin target/release/officegen.exe --expected-version 5.1.0
npm run native:smoke -- --expected-version 5.1.0
npm run native:assets:check -- --version 5.1.0 --include-installers
npm run remediation:check
```

GitHub Actions must build native assets for Linux, macOS, and Windows, upload `install.sh` and `install.ps1`, and verify `.officegen/acceptance/v5.1.0/manifest.json` before publishing the release.
