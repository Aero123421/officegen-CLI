# officegen v5.0.0 Remediation Matrix

This matrix records the v5.0.0 acceptance and review work that must be true before a `v5.0.0` release tag is created.

## Review 1: Agent Contract / False Success

| ID | Area | Finding | Status | Evidence |
| --- | --- | --- | --- | --- |
| V50-001 | Capabilities | Agent default output must remain compact and scoped. | fixed | `capabilities --agent --compact` is implemented and covered by `scripts/v5-acceptance.mjs` A1/A3. |
| V50-002 | Capabilities | MCP and plugin surfaces must not appear in agent-facing JSON. | test-covered | Native smoke and v5 acceptance assert no management runtime sales in capabilities/help. |
| V50-003 | JSON envelope | Unknown commands must fail with structured usage errors. | test-covered | A4 checks `UNKNOWN_COMMAND` and `failureClass: usage`; Rust unit tests cover error envelope. |
| V50-004 | Unsupported preview | Portable PNG must not return fake success. | fixed | A9 checks `view --format png` fails with `FEATURE_NOT_IMPLEMENTED`. |
| V50-005 | Native proof | Native Office proof remains external and disclosed. | limited-disclosed | `verify` returns `NATIVE_PROOF_NOT_RUN`; no native renderer is claimed as default. |
