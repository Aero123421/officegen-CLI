## Release Gates

Required before v3.1.6 tagging:

- `npm run version:check`
- `npm run typecheck`
- `npm test`
- `npm run build`
- `npm run pack:smoke`
- `npm run remediation:check`
- `npm run contract:check`
- `npm run schema:coverage`
- `npm audit --audit-level=moderate`
- `npm run security:fixtures`
- `npm run benchmark:local-fixtures`
- `npm run perfect-spec:evidence`
- `npm run perfect-spec:check -- --gate=publish`
- `npm run release-tarball:smoke -- dist/officegen-v3.1.6.tgz`

Post-tag release completion additionally requires fresh v3.1.6 GitHub install and release-tarball smoke evidence.

## Review 1: Agent Contract / False Success

Reviewed wrapper mutation truthfulness, changed/no-op wrapper evidence, nested edit propagation, JSON stdout cleanliness, and unavailable field diagnostics. Release gates: `npm run contract:check`, `npm run schema:coverage`, `npm test`, and `npm run perfect-spec:check -- --gate=publish`.

## Review 2: Install / Release / Packaging

Reviewed version synchronization, generated dist output, ignored new dist files, GitHub Release tarball packaging, npm publish blocking, and local release tarball smoke. Release gates: `npm run version:check`, `npm run build`, `npm run pack:smoke`, `npm run remediation:check`, and `npm run release-tarball:smoke -- dist/officegen-v3.1.6.tgz`.

## Review 3: Asset / Office Workflow

Reviewed `layout apply` bounds targeting, PPTX object ordinal fallback, semantic geometry diff behavior, DOCX style inventory, and remaining DOCX/XLSX visual renderer limitations. Release gates: full `npm test`, `npm run typecheck`, targeted layout/diff tests, and generated perfect-spec evidence.

## Review 4: UX / Critique / Traceability

Reviewed AI-agent ergonomics for mutation status, JSON parseability, explicit unavailable fields, removal of bigfaster/Cursor-specific repository guidance, and traceability from user-reported v3.1.5 findings to v3.1.6 changes.
