# v4.5.0 Remediation Matrix

Release focus: Rust contract freeze, truthful command registry, MCP/plugin scope removal, fail-closed unsupported surfaces, ZIP/OPC safety before OOXML processing, real schema catalog, compact capabilities, and native binary release gates.

## Review 1: Agent Contract / False Success

| ID | Area | Finding | Status | Evidence |
| --- | --- | --- | --- | --- |
| V45-001 | Command registry | Agent capabilities must be generated from registry metadata, not an unrelated command list. | fixed | `src/registry.rs` defines command status/visibility; runtime capabilities and help read registry views. |
| V45-002 | MCP scope | MCP must not appear in help, capabilities, or agent-visible surfaces. | fixed | `mcp serve` is `RemovedFromScope`, hidden in registry tests, and native smoke rejects exposure. |
| V45-003 | MCP dispatch | `officegen mcp serve` must fail closed with a stable code. | test-covered | Runtime tests and native smoke assert `FEATURE_REMOVED_FROM_SCOPE`. |
| V45-004 | Plugin scope | Plugin commands must not return successful management payloads. | fixed | `plugin install` now fails with `FEATURE_REMOVED_FROM_SCOPE`; native smoke covers it. |
| V45-005 | Agent adapters | Deferred agent install/refresh surfaces must not imply success. | fixed | `agent install` now fails with `FEATURE_NOT_IMPLEMENTED`; runtime and native smoke cover it. |
| V45-006 | PNG preview | Portable raster preview must not write placeholder PNG artifacts. | fixed | `view --format png` fails closed with `FEATURE_NOT_IMPLEMENTED` and no `page-001.png` output. |
| V45-007 | Error catalog | Removed-scope features must be discoverable by agents. | fixed | `errors list/inspect` includes `FEATURE_REMOVED_FROM_SCOPE`. |
| V45-008 | Schema failures | Invalid schema validation must not surface uncataloged `OBJECTIVE_FAILED`. | fixed | Schema validate failures now map to `SCHEMA_INVALID`; schema contract tests cover invalid payloads. |
