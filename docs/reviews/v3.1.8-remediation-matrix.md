# officegen v3.1.8 remediation matrix

Status values: `fixed`, `limited-disclosed`, `test-covered`, `deferred-v3`, `not-applicable`.

| ID | Area | Finding | Status | Evidence |
| --- | --- | --- | --- | --- |

| V31-P002 | PPTX Assets | `asset inspect --embedded` disagreed with verify on chart workbooks | fixed | `packages/formats/src/assets.ts` reports `chartEmbeddedWorkbooks` separately and keeps `embeddedObjects` for true embedded packages only. |
| V31-P003 | PPTX Edit | Large post-edit ZIP size drops lacked package-level evidence | fixed | `packages/formats/src/edit.ts` returns `officegen.packageDiff@1` with part counts, changed/removed/added parts, byte delta, and compression note. |
| V31-P004 | PPTX Preservation | Unknown package parts need regression coverage during edits | test-covered | `packages/formats/tests/formats.test.ts` verifies custom XML parts survive PPTX text edits and removed parts are reported when intentionally dropped. |
| V31-X001 | XLSX Inspect | Scoped `--sheet/--range` inspect had objectMap cells but summary cells/sheets stayed empty | fixed | `packages/formats/src/inspect.ts` derives scoped cell counts and sheet summaries from the scoped objectMap. |
| V31-X002 | XLSX Formula | Existing/variant workbook structures could make visible formula cells fail `xlsx.setFormula` | fixed | `packages/formats/src/ooxml/xlsx.ts` and `packages/formats/src/edit.ts` normalize workbook relationship targets and include formula text in selector matching. |

## Release Gates

Required before v3.1.8 tagging:

- `npm run version:check`
- `npm run typecheck`
- `npm test`
- `npm run build`
- `npm run pack:smoke`
- `npm run remediation:check`
- `npm run contract:check`
- `npm run schema:coverage`
- `npm audit --audit-level=moderate`
- `npm run security:fixtures`
- `npm run benchmark:local-fixtures`
- `npm run perfect-spec:evidence`
- `npm run perfect-spec:check -- --gate=publish`
- `npm run release-tarball:smoke -- dist/officegen-v3.1.8.tgz`

Post-tag release completion additionally requires fresh v3.1.8 GitHub install and release-tarball smoke evidence.

## Review 2: Install / Release / Packaging

Reviewed diagnose/repair edit-op shape, overflow repair de-duplication, chart input validation, Mermaid label parsing, and source-only template candidates.
